من سامان بالاهنگ هستم
توسعه‌دهنده فول‌استک وب با تمرکز بر ایجاد تجربه‌های دیجیتال کاربرپسند و کارآمد. در دنیای وب، هر پروژه برای من فرصتی است برای ترکیب خلاقیت و تکنولوژی. به عنوان توسعه‌دهنده، به معماری تمیز، کدهای قابل نگهداری و عملکرد بهینه اهمیت می‌دهم. از رابط کاربری روان گرفته تا منطق کسب‌وکار در بک‌اند، تمام جنبه‌های یک پروژه وب برایم اهمیت یکسان دارد. با تکنولوژی‌های مدرن وب مانند React، Django و Python کار می‌کنم و همیشه به دنبال یادگیری و به کارگیری بهترین روش‌ها در توسعه وب هستم. برایم مهم است که محصول نهایی نه تنها از نظر فنی قوی باشد، بلکه نیازهای واقعی کاربران را نیز به خوبی برطرف کند. اگر به دنبال توسعه‌دهنده‌ای هستید که هم به زیبایی رابط کاربری و هم به قدرت بک‌اند توجه دارد، خوشحال می‌شوم در مورد پروژه شما صحبت کنیم.

پروژه‌های من

من سالیان سال است که در زمینه طراحی سایت فعالیت میکنم و سایت‌های زیادی طراحی و توسعه داده ام از همکاری در طراحی سایت‌هایی مانند دیگچی، سامان نگار و همکاری در طراحی وبسایت مجلات شرکت نفت و غیره متاسفانه همه این وبسایت ها اکنون در دسترس نیستند ولی بعنوان نمونه کار دوتا از آخرین کارهایم را در زیر مشاهده میکنید.

سری پروژه‌های آموزشی برنامه‌نویسی وب

در این مجموعه، تعدادی پروژه برنامه‌نویسی با هدف یادگیری و آموزش عملی طراحی و توسعه داده شده‌اند. هدف اصلی این پروژه‌ها، آشنایی کاربران با مفاهیم کلیدی توسعه وب و به‌کارگیری تکنولوژی‌های مدرن در دنیای واقعی است. هر پروژه شامل بخش‌های مختلفی از طراحی رابط کاربری تا پیاده‌سازی منطق سمت کاربر و گاهی سمت سرور می‌باشد.

کاربران می‌توانند از این پروژه‌ها برای تمرین، الهام‌گیری و بهبود مهارت‌های خود در زمینه‌ی برنامه‌نویسی استفاده کنند. در این مجموعه، تمرکز ویژه‌ای بر برنامه‌نویسی وب و چارچوب‌های به‌روز آن وجود دارد.

تکنولوژی‌های مورد استفاده در پروژه‌ها

  • HTML5 — ساختاردهی صفحات وب
  • CSS3 — طراحی و استایل‌دهی مدرن
  • JavaScript (ES6+) — منطق و تعاملات کاربر
  • React.js — ساخت رابط‌های کاربری پویا
  • Node.js — پیاده‌سازی بخش بک‌اند و API
  • Express.js — ساخت سرورهای سبک و سریع
  • GSAP — انیمیشن‌های نرم و حرفه‌ای
  • Tailwind CSS — طراحی واکنش‌گرا با سرعت بالا
  • MongoDB — ذخیره و مدیریت داده‌ها
  • Git & GitHub — کنترل نسخه و اشتراک پروژه‌ها
  • python & django — برنامه نویسی بکند
  • pytorch & fastAi — پروژه های هوش مصنوعی
  • adbots — رباتهای افزایش فالور در شبکه های اجتماعی
  • tradebots — رباتهای تریدر
  • wordpress — قالبهای اختصاصی وردپرسی
